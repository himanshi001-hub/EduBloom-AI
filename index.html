    About QuestionAI

    AI-powered exam platform for generating Bloom's Taxonomy-aligned question papers — built for teachers and students.

    The Problem

    Preparing balanced and quality question papers is a time-consuming, repetitive task for teachers. Most papers fail to cover all 6 levels of Bloom's Taxonomy, leading to poor assessment of student learning outcomes.

    Our Solution

    An AI-powered web application where teachers input syllabus topics, question types, difficulty, and marks distribution. Claude AI generates questions mapped to Bloom's levels. Students attempt papers online with live timers and get instant results.

    Tech Stack

    PythonMachine LearningNLP Claude AI / LLMFlask / Django HTMLCSSJavaScript MySQL / MongoDB

    Bloom's Taxonomy Levels

    1. Remember — Recall facts and basic concepts
    2. Understand — Explain ideas or concepts
    3. Apply — Use knowledge in new situations
    4. Analyze — Draw connections, examine parts
    5. Evaluate — Justify a decision or argument
    6. Create — Produce new or original work
